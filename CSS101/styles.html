BLOCK ELEMENT!!!

  1. A BLOCK level element, by default, takes up ALL the space to the right and to the left, internal ti its parent
  2. A BLOCK CAN have height and width
  3. NOTHING can ever be on either side (internal to the parent) because it's 100%.
  4. If you change the width, it will just add margin. Blocks will ignore a margin you add, because they're already 100%
  5. Can be > 100%

INLINE elements!

  1. Respect left and right padding, left and right margin but will ignore top and bottom
  2. Couldn't care less what is left or right.
  3. With a few exceptions (namely images) you CANNOT tell an inline element how big to be.
  4. Will happily ignore all block level rules unless forced to do so

INLINE block

  1. Can have stuff on either side (inline positive)
  2. Can have height/width (block positive)
  3. Can have top/bottom padding/margin (block positive)
